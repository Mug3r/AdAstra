# AdAstra
--- CSE214 2018 Project ---
 
 2D Arcade space shooter wirtten using included Java libraries.
 ========================
 Controls:                                                                               
 ========================
  - A - D: Move Player ship left and right                                                  
  - Spacebar: Shoot                                                                         
  - Left - Right Arrow Keys: Rotate Player ship
