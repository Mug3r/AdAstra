# AdAstra
--- CSE214 2018 Project ---
 =========================================================================================
| A 2D Arcade space shooter wirtten using only included Java libraries, for the most part.|
 =========================================================================================
| Controls:                                                                               |
| A - D: Move Player ship left and right                                                  |
| Spacebar: Shoot                                                                         |
| Left - Right Arrow Keys: Rotate Player ship                                             |
